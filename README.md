📊 E-Commerce Customer Churn Analysis
📌 Overview

This project analyzes customer churn in an e-commerce platform using SQL. The goal is to identify key factors influencing customer attrition and provide actionable insights to improve retention.

🛠️ Tools & Technologies
MySQL
SQL (Joins, Subqueries, Aggregations)


📂 Project Workflow
Data Cleaning
Data Transformation
Exploratory Data Analysis
Business Insights Generation


🔍 Key Analysis
Churned vs Active Customers
Customer Behavior Analysis
Payment Method Preferences
Satisfaction & Complaint Impact
Customer Segmentation


📈 Key Insights
Customers with complaints are more likely to churn
Higher satisfaction leads to better retention
Payment methods influence customer activity
Engagement reduces churn probability
